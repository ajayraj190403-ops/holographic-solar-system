# 🌌 Holographic 3D Universe Exploration Dashboard

An interactive, futuristic 3D web application of the Universe built with **React**, **Three.js**, and **React Three Fiber**. Designed to resemble a highly advanced, sci-fi spaceship navigation dashboard.

## ✨ Features

- **Holographic Aesthetics:** Wireframe rendering, emissive neon materials, and deep space fog.
- **Deep Space Phenomena:** Includes custom 3D mathematical rendering for Black Holes, White Holes, Neutron Stars, and entire Galaxies.
- **Cinematic Camera Tracking:** Click any celestial body to smoothly zoom and lock the camera into its orbit.
- **Dynamic Routing:** Shareable URLs instantly zoom the user to specific cosmic entities (e.g., `/explore/blackhole`).
- **Hyper-Drive Trails:** Dynamic light trails that follow the planets as they move.
- **Educational Database:** A glassmorphism UI panel reveals scientific facts and data when an object is scanned.
- **Time Dilator:** Interactive slider to speed up or slow down orbital mechanics and star spin rates.
- **Instanced Asteroid Belt:** A highly performant rendering of 2,500 individual asteroids.

## 🛠️ Tech Stack

- **Framework:** React.js
- **3D Engine:** Three.js
- **React Abstraction:** @react-three/fiber
- **3D Helpers:** @react-three/drei
- **Deployment:** Vercel

## 🚀 Getting Started

1. Clone the repository:
   git clone https://github.com/ajayraj190403-ops/holographic-universe.git

2. Navigate into the directory and install the dependencies:
   cd holographic-universe
   npm install

3. Start the development server:
   npm run dev
